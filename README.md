## ⚙️ Tech Stack

* **Framework:** [Next.js](https://nextjs.org/)
* **Language:** JavaScript (ES6+)
* **Styling:** CSS with Flexbox & Pseudo Elements
* **Fonts:** [Google Fonts](https://fonts.google.com/) — optimized for fast loading
* **Assets:** SVG-based UI and logo, plus **WEBP** images for ultra-light performance
* **Animations & Logic:** Custom React Hooks for animation and state management
* **Accessibility:** WCAG-compliant design, ARIA roles, and semantic HTML

---

## 🧩 Key Design & Architecture Choices

### ⚡ Fast Loading & Performance

* **Next.js** separates **Server Components** and **Client Components** for efficient rendering and faster delivery.
* **Google Fonts** are optimized and preloaded for minimal layout shifts.
* **WEBP image format** is used exclusively — offering high-quality visuals at a fraction of the file size.

### 🧠 Smart UI Composition

* **SVGs** build complex, scalable UI shapes and the project logo without relying on raster graphics.
* **Flexbox** ensures flexible, responsive layouts across all screen sizes.

### 🎨 Visual Enhancements

* **CSS Pseudo Elements** (`::before`, `::after`) add layers, highlights, and interactive visual effects with no additional DOM overhead.
* **Custom Hooks** provide clean, modular handling of animation timing and global state updates.

### ♿ Accessibility First

* The design adheres to **WCAG 2.1** guidelines for color contrast, text readability, and keyboard navigation.
* **Semantic HTML**, **proper ARIA labeling**, and **focus management** are applied throughout to ensure inclusive interaction for all users.
